Detailed Description

Represents a folder that can contain notes and other folders.

The Folder class provides hierarchical organization for notes with support for unlimited nesting depth. Folders can contain both notes and subfolders, allowing for complex organizational structures. Each folder maintains its own metadata and provides comprehensive management capabilities.

Key features:

  • Automatic unique ID generation
  • Support for nested folder structures (unlimited depth)
  • Note management with search capabilities
  • Subfolder management with search capabilities
  • Recursive operations (e.g., total note counting)
  • Tree visualization support
