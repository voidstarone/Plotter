Detailed Description

Represents a note with content and arbitrary attributes.

The Note class provides a flexible container for storing textual content along with arbitrary metadata attributes. Each note maintains automatic timestamps, supports type-safe attribute storage, and provides comprehensive content management capabilities.

Key features:

  • Automatic unique ID generation
  • Automatic creation and modification timestamps
  • Type-safe arbitrary attribute system using std::any
  • Template-based attribute access with type checking
  • Support for any C++ type as attribute values
  • Automatic timestamp updates on content changes
