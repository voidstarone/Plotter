Detailed Description

Represents a top-level project containing folders and organizing notes.

The Project class is the root container in the hierarchical note-taking system. It manages a collection of folders, provides search functionality, and maintains project metadata including name, description, and unique identification.

Key features:

  • Automatic unique ID generation
  • Folder management with search capabilities
  • Tree visualization support
  • Memory management through smart pointers
